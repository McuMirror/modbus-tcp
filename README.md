# Modbus TCP

Modbus TCP client/server implementation in C# with DotNetty.

A port of [modjn](https://github.com/klymenek/modjn).

## Currently implemented modbus functions

### Client

+ 0x01: Read Coils
+ 0x02: Read Discrete Inputs
+ 0x03: Read Holding Registers

### Server

+ 0x01: Read Coils
+ 0x02: Read Discrete Inputs
+ 0x03: Read Holding Registers
